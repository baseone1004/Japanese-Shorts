# assets

## character.png ← 여기에 채널 캐릭터를 넣으세요

이 폴더에 **`character.png`** 라는 이름으로 채널 고정 캐릭터 이미지를 저장하면,
④ 이미지 탭에서 생성하는 모든 이미지가 이 캐릭터를 참조해 같은 얼굴·같은 화풍으로 그려집니다.

```
assets/character.png
```

- 형식: PNG
- 배경은 있어도 되고 투명이어도 됩니다. 캐릭터가 크고 선명하게 보이는 이미지일수록 일관성이 좋습니다.
- 정사각형(예: 1024×1024) 권장. 생성되는 이미지는 이것과 무관하게 항상 세로 9:16로 나옵니다.

파일이 없으면 이미지 탭에서 "캐릭터 이미지를 찾지 못했습니다" 안내가 뜹니다.
캐릭터 없이 배경/소품만 생성하고 싶으면 `캐릭터 고정` 체크를 해제하면 됩니다.

> 이 파일(`character.png`)은 `.gitignore`에 포함돼 있어 GitHub에 올라가지 않습니다.
> 레포가 public이라 캐릭터 원본을 공개하지 않기 위한 기본값입니다.
> 공개해도 상관없다면 `.gitignore`에서 `assets/character.png` 줄을 지우세요.
