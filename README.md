# 🇯🇵 일본 쇼츠 스튜디오 (Japanese-Shorts)

일본 유튜브 쇼츠를 만들기 위한 로컬 웹 도구. Claude API(`claude-opus-4-8`)를 호출해
**대본 생성 → 번역 → 메타데이터**를 한 흐름으로 처리한다.

**장르별 톤 프리셋 12종** — 요리/여행/리뷰/이슈/동물/꿀팁/뷰티/운동/재테크/K-POP/유머/일반.
한 번 고르면 세 모드 전부에 일본어 말투·훅 표현·해시태그 스타일이 일관되게 적용된다.

## 세 가지 모드

### ① 대본 생성

**`주제 10개 추천받기`** 를 누르면 선택한 카테고리에 맞는 촬영 가능한 주제 10개가 카드로 나온다.
각 카드에는 일본어 훅 문장, 먹히는 이유, 촬영 난이도(쉬움/보통/어려움)가 붙는다.
**카드를 누르면 그 주제로 대본 생성까지 바로 이어진다.**

주제를 직접 입력해도 되고, **비워두면 해당 장르에서 잘 먹힐 주제를 AI가 직접 정한다.**

- 첫 3초 훅 필수, 타임라인 자동 구성
- 한 행 = 화면 자막 한 장, 일본어에 `/` 끊어읽기 삽입
- 각 행에 한국어 해석 병기 (검수용) + 촬영·편집 제작 메모
- 결과를 **②번이나 ③번 탭으로 바로 넘기는 버튼** 제공

### ② 대본 번역

한국어(또는 외래어) 대본 → 일본 현지 구어체 번역.

- 섬네일 제목 3안 (`/` 호흡 분할, 일본어 + 한국어 병기), 설명문, 해시태그
- 번역 표는 의미 단위로 행 분리
  - 원문이 한국어면 `[타임라인 | 원문 | 일본어]` 3열
  - 원문이 외래어면 `[타임라인 | 원문 | 한국어 | 일본어]` 4열

### ④ 이미지 프롬프트

대본의 각 컷을 그릴 **이미지 생성 프롬프트**를 만든다. 이미지를 직접 만들지는 않는다 —
나온 영문 프롬프트를 원하는 그림 도구(ChatGPT, Midjourney, nano banana 등)에 붙여넣어 쓴다.

- **캐릭터 고정 묘사**가 모든 컷 프롬프트 맨 앞에 동일하게 들어가 캐릭터가 흔들리지 않는다.
  화면에서 수정 가능하고 브라우저에 저장된다.
- 세로 9:16, 상단 자막 여백 확보, 이미지 내 글자 배제가 프롬프트에 자동 포함된다.
- 엑셀 복사 / CSV 지원.

### ⑤ 사운드

대본에 맞는 **BGM 프롬프트**(Suno용)와 **컷별 효과음 큐**를 만든다.

- BGM은 서로 다른 방향 2안 (예: 긴장감형 / 밝고 경쾌형). 영문 스타일 프롬프트 + 권장 BPM.
  가사 없는 instrumental로 지시되며, 자막을 방해하지 않도록 설계된다.
- 효과음은 **정말 필요한 지점에만** 넣도록 지시했다 (과하면 싸구려로 들린다).
  캡컷 한국어 검색어와 영어 검색어를 함께 준다.
- 믹싱 메모 (BGM 볼륨, 정답 공개 전 음악 멈추기 등) 포함.

> 🎵 **Suno는 음악 생성 AI라 "쿵", "띠링" 같은 단발 효과음은 만들지 못한다.**
> BGM 프롬프트만 Suno에 넣고, 효과음은 검색어로 편집 프로그램 라이브러리에서 찾을 것.

### ＋ 카테고리 발굴

기본 13개 말고 새로 파볼 만한 장르를 제안받는다. 마음에 들면 버튼 한 번으로 그 카테고리가
드롭다운에 추가되고 바로 주제·대본을 뽑을 수 있다 (브라우저에 저장되어 다음에도 남는다).

> ⚠️ **AI는 실시간 유튜브 순위를 볼 수 없다.** 제안은 실제 트렌드 데이터가 아니라 구조적
> 근거에 기반한 가설이다. 각 제안의 확신도(confidence)와 약점(risk)을 함께 표시하니
> 참고하되, 최종 판단은 직접 유튜브에서 검색해 확인할 것.

### ③ 메타데이터

대본이나 **영상 내용 설명 텍스트**를 넣으면 유튜브 메타데이터를 뽑는다.

- 일본어 제목 1개 + 한국어 해석
- 일본어 설명문 + 한국어 해석
- 태그 정확히 5개 (일본어 / 한국어 해석 병기)
- 지정 양식 그대로 전체 복사

## 공통

- **엑셀용 TSV 복사 / CSV 다운로드** (UTF-8 BOM 포함 — Excel에서 한글·일본어 안 깨짐)
- 모든 모드에 적용되는 **추가 지시** 입력란 (예: "존댓말로", "20대 여성 타겟")

## 사용법

| 파일 | 하는 일 |
| --- | --- |
| **`start.vbs`** | 실행 — 검은 창 없이 앱 창으로 열립니다 |
| `stop.vbs` | 서버 종료 (평소엔 쓸 일 없음) |
| `start.bat` | 문제가 생겼을 때 오류를 확인하는 용도 (검은 창이 뜹니다) |

**`start.vbs` 를 더블클릭**하면 검은 콘솔 창 없이 주소창도 없는 앱 창으로 뜹니다
(Chrome 또는 Edge의 앱 모드). 서버는 뒤에서 조용히 돌아갑니다.

**따로 끌 필요는 없습니다.** 앱 창을 닫아도 서버는 남지만, 버튼을 눌러야만 API를
호출하므로 켜둬도 비용이 발생하지 않고 메모리도 50~80MB 수준이다. PC를 끄면 사라진다.
이미 켜져 있으면 `start.vbs`는 서버를 새로 띄우지 않고 창만 연다.

프로그램이 이상하게 동작해서 껐다 켜야 할 때만 `stop.vbs`를 쓰면 된다.

### 바탕화면 바로 가기

바탕화면의 **`일본 쇼츠 스튜디오`** 를 더블클릭하면 실행됩니다.

바로 가기가 사라졌거나 다시 만들려면 PowerShell에서:

```powershell
$d = [Environment]::GetFolderPath('Desktop'); $p = 'C:\projects\Japanese-Shorts'
$sh = New-Object -ComObject WScript.Shell
$s = $sh.CreateShortcut("$d\일본 쇼츠 스튜디오.lnk")
$s.TargetPath = "$p\start.vbs"; $s.WorkingDirectory = $p; $s.IconLocation = "$p\icon.ico,0"; $s.Save()
```

> 프로그램 파일(`start.vbs` 등) 자체를 바탕화면으로 **옮기면 안 됩니다.**
> 같은 폴더에 있어야 동작합니다. 바로 가기만 옮기세요.

처음 한 번은 `.env` 파일에 API 키를 넣어야 합니다 (아래 설치 항목 참고).

## 설치

```bash
npm install
cp .env.example .env    # Windows: copy .env.example .env
```

`.env`에 [Anthropic Console](https://console.anthropic.com)에서 발급받은 키를 넣는다:

```
ANTHROPIC_API_KEY=sk-ant-...
```

## 실행

```bash
npm start
```

→ http://localhost:3000

## 입력 예시 (② 번역)

```
00:00 안녕하세요, 오늘은 3분만에 만드는 김치볶음밥 알려드릴게요
00:04 재료는 딱 네 개면 됩니다
00:07 신김치, 찬밥, 계란, 그리고 참기름
```

타임라인은 없어도 동작한다 (타임라인 열이 빈 칸으로 나온다).

## 구조

| 파일 | 역할 |
| --- | --- |
| `server.js` | Express 서버 + Claude API 호출 (`/api/topics`, `/api/script`, `/api/translate`, `/api/metadata`) |
| `prompt.js` | 모드별 시스템 프롬프트 + 구조화 출력 JSON 스키마 |
| `categories.js` | 장르별 톤 프리셋 정의 |
| `public/` | 프론트엔드 (바닐라 JS, 탭 3개) |

## 카테고리 추가하기

`categories.js`의 `CATEGORIES` 객체에 항목을 하나 추가하면 끝이다.
서버 재시작 후 드롭다운에 자동으로 나타난다.

```js
newGenre: {
  label: '새 장르',
  tone: '어떤 일본어 문체를 쓸지',
  hooks: ['훅 표현1', '훅 표현2'],
  tags: ['#해시태그'],
  notes: '현지화 주의사항',
},
```

## 참고
